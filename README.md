## Cómo ejecutar los proyectos

### Para proyectos de C++ (Universidad):
1. Abrir el archivo `.cpp` o el proyecto `.dev` en **Dev-C++**.
2. Asegurarse de tener configurado el estándar `ISO C++11` en *Compiler Options*.
3. Compilar y ejecutar con `F11`.

### Para proyectos de Java (Universidad):
1. Abrir la carpeta del proyecto en **IntelliJ IDEA**.
2. Configurar el SDK de Java (JDK 11 o superior recomendado).
3. Ejecutar la clase principal (`Main`).
