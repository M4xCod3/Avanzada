# Avanzada
Repositorio con ejercicios, proyectos y desarrollos personales enfocados en programación avanzada, estrucutra de datos y buenas practicas de desarrollo.


###  Universidad
Proyectos y ejercicios desarrollados durante mi formación académica, enfocados en fundamentos sólidos de programación y análisis algorítmico 

* **C++ (Programación Avanzada)**
    * **Entorno:** Dev-C++ 5.11 (Compilador TDM-GCC 4.9.2).
    * **Paradigma:** Programación orientada a objetos (POO).
    * **Temas trabajados:** 
        * Herencia y polimorfismo.
        * Gestión dinamica de memoria.
        * manejo de archivos
        * modularización de código.
* **Java (Estructura de Datos y Algoritmos)**
    * **Entorno:** IntelliJ IDEA.
    * **Enfoque:** Implementación de estructuras lineales y no lineales (Listas, Pilas, Árboles, Grafos) y análisis de complejidad (Big O).
 
### Personal
Proyectos propios donde exploro conceptos avanzados y nuevas tecnologías.

## Cómo ejecutar los proyectos

### Para proyectos de C++ (Universidad):
1. Abrir el archivo `.cpp` o el proyecto `.dev` en **Dev-C++**.
2. Asegurarse de tener configurado el estándar `ISO C++11` en *Compiler Options*.
3. Compilar y ejecutar con `F11`.

### Para proyectos de Java (Universidad):
1. Abrir la carpeta del proyecto en **IntelliJ IDEA**.
2. Configurar el SDK de Java (JDK 11 o superior recomendado).
3. Ejecutar la clase principal (`Main`).
